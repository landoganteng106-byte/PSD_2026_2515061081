Judul : Pengelolaan member gym

Deskripsi :
Sistem ini adalah program yang dirancang untuk mengelola data member gym secara efisien menggunakan struktur data Binary Search Tree. Penggunaan BST memungkinkan sistem untuk melakukan pencarian data dengan kecepatan yang jauh lebih tinggi dibandingkan dengan daftar biasa (array/list), 
terutama ketika jumlah member sudah sangat banyak.

![img alt](https://cdn.corenexis.com/files/c/6847885720.png)

kode ke-1,berguna untuk mendefinisikan kelas untuk menyimpan data tiap member.
kode ke-2,berguna sebagai konstruktor untuk inisialisasi objek.
kode ke-3,berguna untuk menyimpan ID unik member sebagai kunci pencarian.
kode ke-4,berguna untuk menyimpan nama member.
kode ke-5, berguna sebagai penunjuk ke anak kiri
kode ke-6, berguna sebagai Penunjuk ke anak kanan
kode baris ke-8 berguna untuk sebagai kelas utama untuk mengelola sistem.
kode baris ke-9 berguna untuk Inisialisasi sistem
kode baris ke-10 menandai bahwa akar pohon dimulai dari None
kode baris ke-12 membuat  untuk menambah member baru.
kode baris ke-13 mengecek apakah rootnya sudah ada,jika tidak pada baris ke-14 membuat root jika rootnya sebelumnya tidak ada
pada baris ke-15,jika rootnya sudah ada,data yang diinput akan dijadikan node baru atau member
pada baris ke-19 ,dibuat fungsi untuk memasukkan data atau insert
